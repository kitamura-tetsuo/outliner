# User Guide: Project Management

This guide provides a complete overview of the project management features available in the application.

## How to Rename a Project

1. **Go to Project Settings:** Open the project you wish to rename and navigate to the "Settings" tab.
2. **Find the Rename Option:** Locate the "Rename Project" button or input field.
3. **Enter the New Name:** Type the new name for your project and click "Save" or "Rename."

The project title will be updated across the application, including in your project list and breadcrumb navigation.

## How to Manage Permissions

You can invite collaborators and manage their access levels to ensure your projects are secure.

1. **Open Access Control:** In the project settings, go to the "Permissions" or "Access Control" section.
2. **Invite Users:** Click "Invite" and enter the email address of the person you want to add.
3. **Assign Roles:** Assign a role to the user, such as:
   - **Editor:** Can view and modify the project.
   - **Viewer:** Can only view the project.
4. **Revoke Access:** You can remove a user's access at any time by clicking the "Remove" or "Revoke" button next to their name.

## How to Share Projects Publicly

You can create a public, read-only link to share your project with a wider audience.

1. **Enable Public Sharing:** In the project settings, find the "Public Sharing" section.
2. **Generate Link:** Click the button to generate a unique public link.
3. **Set a Password (Optional):** For added security, you can set a password that viewers must enter to access the project.
4. **Disable Link:** You can disable the public link at any time to make the project private again.

## How to Delete and Restore Projects

If you no longer need a project, you can delete it. Deleted projects can be restored later if needed.

1. **Delete a Project:** In the project settings, click the "Delete Project" button. The project will be moved to the "Trash" or "Archived" folder.
2. **View Deleted Projects:** To see your deleted projects, navigate to the "Trash" section from your main dashboard.
3. **Restore a Project:** Find the project you want to restore and click the "Restore" button. It will be moved back to your active projects list.
4. **Permanently Delete:** To permanently delete a project, open the "Trash," select the project, and click "Delete Permanently." This action cannot be undone.
