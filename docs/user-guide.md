# User Guide

This guide provides instructions on how to use the various features of this application.

## Project Management

### How to Rename a Project

1. Navigate to the project management page.
2. Locate the project you want to rename in the project list.
3. Click the "Rename" button next to the project name.
4. Enter the new project name in the dialog that appears and click "Save".

### How to Manage Permissions

1. From the project management page, click the "Permissions" button for the desired project.
2. To invite a new user, enter their email address and click "Invite".
3. To change an existing user's permissions, select the desired permission level (e.g., read-only, read-write) from the dropdown menu next to their name.

### How to Share Projects Publicly

1. On the project management page, toggle the "Public Sharing" switch for the project you want to share.
2. A unique, read-only link will be generated.
3. Copy the link and share it with anyone you want to have view-only access to the project.

### How to Delete and Restore Projects

1. To delete a project, click the "Delete" button next to the project on the project management page. The project will be moved to the trash.
2. To restore a project from the trash, navigate to the trash view, locate the project, and click the "Restore" button.
3. To permanently delete a project, click the "Permanently Delete" button next to the project in the trash view.
